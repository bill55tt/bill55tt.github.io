使用效果

配置完成后，你的文章页面将会：

  1. 在侧边栏显示文章目录
  2. 目录会自动根据文章标题生成
  3. 可以点击目录直接跳转到对应位置
  4. 当前阅读位置的目录项会高亮显示

注意事项

  1. 确保你的文章使用了正确的 Markdown 标题格式（# ## ###）
  2. 目录只会显示在文章页面
  3. 如果文章没有标题，则不会显示目录
  4. 建议保持标题层级的连续性，不要跳级使用

常见问题

目录不显示？

检查以下几点：

  1. 确认 _config.next.yml 中的 toc.enable 设置为 true
  2. 确认文章 Front-matter 中设置了 toc: true
  3. 确认文章中使用了正确的标题格式

目录显示不全？

  1. 检查 max_depth 设置是否合适
  2. 确认标题层级使用正确
  3. 检查标题格式是否规范

目录不能点击跳转？

  1. 确认主题版本是否最新
  2. 检查是否有 JavaScript 错误
  3. 尝试清除浏览器缓存
