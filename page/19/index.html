Application Examples of Parallel Circuits (3-9)

Key Points

  • Ammeter Structure: Composed of a galvanometer and internal resistance RA in series; the lower the internal resistance, the better, ideally zero
  • Current Measurement: The ammeter must be connected in series with the component being measured
  • Multiplier (Shunt Resistor): To extend the ammeter’s measuring range, a low-resistance resistor RS is connected in parallel, called a shunt resistor
  • Multiplier Formula:
    [ n = \frac{I_T}{I_A} = 1 + \frac{R_A}{R_S} ]
  • Shunt Resistor Calculation:
    [ R_S = \frac{R_A}{n - 1} ]
  • Examples:
    • Ammeter full scale 10mA, internal resistance 50Ω, to measure 1A, parallel a 0.5Ω shunt resistor
    • If RS is 1/9 of RA, the measuring range is extended 10 times
  • Parallel Appliances Rated Voltage Judgment:
    • Appliances with matching rated voltage work normally (e.g., computer, refrigerator at 110V)
    • Appliances with rated voltage lower than supply voltage will be damaged (e.g., 24V intercom connected to 110V)
    • Appliances with rated voltage higher than supply voltage cannot operate properly (e.g., 220V air conditioner connected to 110V)

並聯電路應用實例（3-9）

重點歸納

  • 電流表結構：由檢流計與內電阻RA串聯組成，內電阻越低越好，理想為零
  • 測量電流：電流表必須與被測元件串聯
  • 倍增器（分流器）：為擴大電流表測量範圍，並聯一個低阻值電阻RS，稱為分流器
  • 倍增率公式：
    [ n = \frac{I_T}{I_A} = 1 + \frac{R_A}{R_S} ]
  • 分流器電阻計算：
    [ R_S = \frac{R_A}{n - 1} ]
  • 實例：
    • 電流表滿刻度10mA，內阻50Ω，要測量1A，需並聯0.5Ω分流器
    • 若RS是RA的1/9，測量範圍擴大為10倍
  • 並聯電器額定電壓判斷：
    • 額定電壓相符可正常工作（如電腦、冰箱110V）
    • 額定電壓低於供電電壓會燒毀（如24V對講機接110V）
    • 額定電壓高於供電電壓無法正常運作（如220V冷氣接110V）

2-5.8 RC Charging Circuit Transient Characteristics / 2-5.8 RC充電電路暫態特性與時間常數

2-5.8 RC Charging Circuit Transient Characteristics

Key Points

RC Circuit Structure

  • Capacitor and resistor in series
  • Charging starts when switch closes
  • Basic configuration for transient analysis

Initial Conditions

  • At switch closure:
    • Capacitor acts as short circuit
    • Voltage across capacitor = 0
    • Maximum current: (I_{\max} = \frac{E}{R})

Charging Process

  • Capacitor voltage (V_C(t)) increases over time
  • Resistor voltage (V_R(t)) decreases over time
  • Current (i(t)) decreases over time
  • Exponential relationship in all changes

Fully Charged State

  • Capacitor voltage equals source voltage (E)
  • Resistor voltage becomes 0
  • Current becomes 0
  • Circuit behaves as open circuit

Time Constant (\tau)

  • Defined as: ( \tau = R \times C )
  • Benchmark for transient duration
  • Larger (\tau) = longer charging time
  • Smaller (\tau) = shorter charging time

2-5.8 RC充電電路暫態特性與時間常數

重點歸納

RC充電電路結構

  • 電容器與電阻串聯
  • 開關閉合時開始充電
  • 暫態分析的基本配置

瞬間特性

  • 開關剛閉合時：
    • 電容器等效短路
    • 電容兩端電壓為0
    • 電流最大：(I_{\max} = \frac{E}{R})

充電過程

  • 電容電壓 (V_C(t)) 隨時間增加
  • 電阻電壓 (V_R(t)) 隨時間減少
  • 電流 (i(t)) 隨時間減少
  • 所有變化呈指數關係

充飽狀態

  • 電容電壓達到電源電壓 (E)
  • 電阻兩端電壓為0
  • 電流為0
  • 電路呈斷路狀態

時間常數 (\tau)

  • 定義為：( \tau = R \times C )
  • 表示暫態持續時間的基準
  • 時間常數越大，充電時間越長
  • 時間常數越小，充電時間越短

2-5.9 RC Charging Circuit Transient Current Formula / 2-5.9 RC充電電路暫態電流公式解析

2-5.9 RC Charging Circuit Transient Current Formula

Key Points

Initial Conditions

  • Capacitor connected to DC circuit
  • Switch S closed
  • Initial capacitor voltage = 0
  • Maximum current: ( i(0) = \frac{E}{R} )

Circuit Analysis

  • Following Kirchhoff’s voltage law:
    [
    E = V_R(t) + V_C(t)
    ]
  • Voltage relationships:
    • Resistor voltage: ( V_R(t) = i(t) \times R )
    • Capacitor voltage: ( V_C(t) = \frac{Q(t)}{C} )
    • Charge: ( Q(t) = \int i(t) dt )

Current Formula

  • Derived differential equation solution:
    [
    i(t) = \frac{E}{R} e^{-\frac{t}{RC}}
    ]
  • Formula components:
    • ( \frac{E}{R} ): Initial current at switch closure
    • ( t ): Time
    • ( RC ): Time constant
    • ( e ): Euler’s number (~2.71828)

Time Characteristics

  • Current decays exponentially
  • Approaches zero at ( 5 \times RC )
  • Circuit stabilizes at full charge

Common Exponential Values

  • ( e^{-1} = 0.368 )
  • ( e^{-2} = 0.135 )
  • ( e^{-3} = 0.05 )
  • ( e^{-4} = 0.018 )
  • ( e^{-5} \approx 0 )

2-5.9 RC充電電路暫態電流公式解析

重點歸納

初始條件

  • 電容器接入直流電路
  • 開關S閉合
  • 電容器初始電壓為0
  • 電流最大值：( i(0) = \frac{E}{R} )

電路分析

  • 根據克希荷夫電壓定理：
    [
    E = V_R(t) + V_C(t)
    ]
  • 電壓關係：
    • 電阻電壓：( V_R(t) = i(t) \times R )
    • 電容電壓：( V_C(t) = \frac{Q(t)}{C} )
    • 電量：( Q(t) = \int i(t) dt )

電流公式

  • 微分方程解：
    [
    i(t) = \frac{E}{R} e^{-\frac{t}{RC}}
    ]
  • 公式組成：
    • ( \frac{E}{R} )：開關閉合瞬間的初始電流
    • ( t )：時間
    • ( RC )：時間常數
    • ( e )：自然指數（約2.71828）

時間特性

  • 電流呈指數衰減
  • 約在 ( 5 \times RC ) 時趨近於0
  • 電容充電完成，電路穩定

常用自然指數值

  • ( e^{-1} = 0.368 )
  • ( e^{-2} = 0.135 )
  • ( e^{-3} = 0.05 )
  • ( e^{-4} = 0.018 )
  • ( e^{-5} \approx 0 )

2-5.10 RC Charging Circuit Transient Formula Review

Key Points

Transient Changes

  • In RC charging circuit:
    • Resistor voltage changes
    • Capacitor voltage changes
    • Circuit current changes
  • All changes are time-dependent

Mathematical Description

  • Changes described by formulas
  • Natural exponential function: ( e^{-\frac{t}{RC}} )
  • Time constant: ( \tau = RC )

Voltage and Current Formulas

  • Capacitor voltage ( V_C(t) ):

    • Increases over time
    • Formula: ( 1 - e^{-\frac{t}{RC}} )

  • Resistor voltage ( V_R(t) ):

    • Decreases over time
    • Formula: ( e^{-\frac{t}{RC}} )

  • Circuit current ( i(t) ):

    • Decreases over time
    • Formula: ( e^{-\frac{t}{RC}} )

Time Characteristics

  • Time constant ( \tau = RC )
  • Transient disappears at ( 5\tau )
  • Circuit reaches steady state

重點歸納

暫態變化

  • RC充電電路中：
    • 電阻兩端電壓變化
    • 電容兩端電壓變化
    • 迴路電流變化
  • 所有變化都與時間相關

數學描述

  • 變化可用數學公式描述
  • 自然指數函數：( e^{-\frac{t}{RC}} )
  • 時間常數：( \tau = RC )

電壓與電流公式

  • 電容電壓 ( V_C(t) )：

    • 隨時間增加
    • 公式：( 1 - e^{-\frac{t}{RC}} )

  • 電阻電壓 ( V_R(t) )：

    • 隨時間減少
    • 公式：( e^{-\frac{t}{RC}} )

  • 迴路電流 ( i(t) )：

    • 隨時間減少
    • 公式：( e^{-\frac{t}{RC}} )

時間特性

  • 時間常數 ( \tau = RC )
  • 約在 ( 5\tau ) 時暫態消失
  • 電路進入穩定狀態

2-5.1 Resistance Temperature Coefficient Examples

Key Points

  1. Definition of Temperature Coefficient of Resistance (TCR)
    The TCR (α) represents the fractional change in resistance per degree Celsius increase in temperature, calculated by:
    $$
    \alpha_{t1} = \frac{R_2 - R_1}{(t_2 - t_1) R_1}
    $$
    where (R_1) and (R_2) are the resistances at temperatures (t_1) and (t_2) respectively.

  2. Example Calculations

    • At 20°C, resistance is 2Ω; at 220°C, resistance is 4Ω; TCR at 20°C is 0.005.
    • Resistance at other temperatures can be found by:
      $$
      R_2 = R_1 \times [1 + \alpha_{t1} (t_2 - t_1)]
      $$
      For example, resistance at 200°C is 3.8Ω.

  3. Positive Temperature Coefficient
    Resistance increases with temperature, known as positive temperature coefficient.

  4. TCR Variation at High Temperatures
    Use a modified formula to calculate TCR at high temperatures (e.g., 2500°C).

  5. Copper Wire Resistance Calculation
    Calculate resistance at different temperatures using copper’s absolute temperature property.

2-5.1 電阻溫度係數範例練習

重點歸納

  1. 電阻溫度係數定義
    電阻溫度係數（α）表示溫度每升高1°C時，電阻值增加的比例，計算公式為：
    $$
    \alpha_{t1} = \frac{R_2 - R_1}{(t_2 - t_1) R_1}
    $$
    其中，(R_1)和(R_2)分別是溫度(t_1)和(t_2)時的電阻值。

  2. 範例計算

    • 例題中，20°C時電阻為2Ω，220°C時電阻為4Ω，計算20°C時的電阻溫度係數為0.005。
    • 利用電阻溫度係數計算其他溫度下的電阻值，公式為：
      $$
      R_2 = R_1 \times [1 + \alpha_{t1} (t_2 - t_1)]
      $$
      例如，計算200°C時的電阻值為3.8Ω。

  3. 正溫度係數特性
    電阻隨溫度升高而增加，稱為正溫度係數。

  4. 高溫下電阻溫度係數變化
    利用變形公式計算高溫（如2500°C）時的電阻溫度係數。

  5. 銅線電阻計算
    利用銅的絕對溫度特性計算不同溫度下的電阻值。
