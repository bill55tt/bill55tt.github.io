High School Basic Electricity: Resistance Basics - 高中基本電學：電阻基礎

Posted on 2025-05-05 Edited on 2025-05-24 In Electronics , Physics , Basic Electronics Views: Disqus:

Introduction

This article explores the fundamental concepts of electrical resistance, including its definition, factors affecting resistance, mathematical formulas, and practical applications in electrical engineering.

引言

本文探討電阻的基本概念，包括其定義、影響因素、數學公式以及在電氣工程中的實際應用。

Key Points

  1. Definition of Resistance:
    Resistance is the opposition encountered by electric current flowing through a material, measured in Ohms (Ω).
  2. Factors Affecting Resistance:
    Resistivity (ρ): The resistance capability of different materials.
    Length (L): Longer lengths result in higher resistance.
    Cross-sectional Area (A): Larger areas lead to lower resistance.
  3. Mathematical Formula:
    The formula for calculating resistance is:
    $$R = \frac{\rho \cdot L}{A}$$
    Where R is resistance, ρ is resistivity, L is length, and A is cross-sectional area.
  4. Relationship Between Resistance and Conductance:
    Conductance is the reciprocal of resistance, indicating the ability to conduct electricity.
  5. Practical Applications:
    Engineers use resistors to control current in circuits, ensuring stable operation.

重點歸納

  1. 電阻的定義：
    電阻是電流在物體中流動時所遇到的阻力，單位為歐姆（Ω）。
  2. 電阻的影響因素：
    電阻係數（ρ）：不同材質的電阻能力。
    物體長度（L）：長度越長，電阻越大。
    截面積（A）：截面積越大，電阻越小。
  3. 數學公式：
    電阻的計算公式為：
    $$R = \frac{\rho \cdot L}{A}$$
    其中，R為電阻，ρ為電阻係數，L為長度，A為截面積。
  4. 電阻與電導的關係：
    電導是電阻的倒數，表示導電能力。
  5. 實際應用：
    工程師利用電阻來控制電路中的電流，確保穩定操作。

2-5.3 How to Read Color Code Resistors

Key Points

Resistor Value Indication Methods

  • Resistors indicate resistance values by two methods:
    • Color code (for small resistors)
    • Numeric code (for larger resistors)

Color Code Classification

  • Color code resistors are classified by tolerance:
    • 3-band resistors (±20% tolerance)
    • 4-band resistors (±5% tolerance)
    • 5-band resistors (precision resistors)

Color Code Meaning

  • Colors represent:
    • Digits (first bands)
    • Multipliers (middle bands)
    • Tolerance (last band)
  • Example: Brown represents:
    • Digit: 1
    • Multiplier: 10¹
    • Tolerance: ±1%

Reading Color Codes

  • Reading process:
    1. First bands: represent digits
    2. Middle bands: represent multiplier
    3. Last band: represents tolerance

Numeric Code System

  • Used mainly for high-power resistors
  • Indicates:
    • Power rating
    • Resistance value
    • Tolerance

2-5.3 色碼電阻器的判讀方法

重點歸納

電阻值標示方法

  • 電阻元件上標示電阻值有兩種方法：
    • 色碼標示法（小體積電阻）
    • 數字標示法（大體積電阻）

色碼分類

  • 色碼標示法依誤差範圍分為：
    • 三環（誤差±20%）
    • 四環（誤差±5%）
    • 五環（精密電阻）

色碼意義

  • 色碼代表：
    • 數字（前幾環）
    • 倍率（中間環）
    • 誤差（最後一環）
  • 例如：棕色代表：
    • 數字：1
    • 倍率：10¹
    • 誤差：±1%

色碼判讀方法

  • 判讀步驟：
    1. 前幾環：代表數字
    2. 中間環：代表倍率
    3. 最後一環：代表誤差

數字標示法

  • 多用於大功率電阻
  • 標示內容：
    • 功率
    • 阻值
    • 誤差

2-5.2 Introduction and Characteristics of Common Resistors

Key Points

Resistor Selection Considerations

  • When selecting resistors, consider specifications such as rated power, tolerance, operation mode, and material.
  • Rated power: Maximum power the resistor can handle; larger size means higher power capacity.
  • Tolerance: Possible deviation in resistance value, affecting overall circuit accuracy.

Operation Modes

  • Operation modes include fixed resistors, variable resistors, and semi-variable resistors.

Fixed Resistor Types by Material

  1. Carbon Composition Resistors

    • Low cost
    • Large tolerance
    • Suitable for low power applications

  2. Carbon Film Resistors

    • Smaller tolerance
    • Low power applications

  3. Wire-wound Resistors

    • High power capacity
    • Low tolerance
    • Higher cost

  4. Cement Resistors

    • Good heat resistance
    • Suitable for high power and low tolerance applications

  5. Chip Resistors

    • Small size
    • Used in 3C products

  6. Metal Film Resistors

    • High power capacity
    • Low tolerance
    • High stability

Variable Resistors

  • Variable Resistors (VR)

    • Manually adjustable resistance
    • Commonly used for volume or light dimming

  • Semi-variable Resistors (SVR)

    • Adjusted with tools
    • Usually operated by professionals

2-5.2 常用電阻器介紹與特性解析

重點歸納

電阻器選擇考量

  • 電阻器選擇需考慮規格，包括額定功率、誤差值、工作方式與材質。
  • 額定功率：電阻器能承受的最大功率，體積越大功率越高。
  • 誤差值：電阻值的可能偏差，影響整體電路精度。

工作方式

  • 工作方式分為固定電阻、可變電阻與半固定可變電阻。

固定電阻材質分類

  1. 碳質電阻

    • 成本低
    • 誤差大
    • 適用小功率

  2. 碳膜電阻

    • 誤差較小
    • 小功率應用

  3. 線繞電阻

    • 高功率
    • 誤差小
    • 成本較高

  4. 水泥電阻

    • 耐熱性好
    • 適合高功率低誤差應用

  5. 晶片電阻

    • 體積小
    • 適用3C產品

  6. 金屬膜電阻

    • 高功率
    • 低誤差
    • 高穩定性

可變電阻

  • 可變電阻（VR）

    • 可手動調整電阻值
    • 常用於音量或燈光調節

  • 半固定可變電阻（SVR）

    • 需工具調整
    • 通常由專業人員操作

Basic Electricity 1 - Section 3-8: Current Division Rule

Key Points

  • Current Division Rule: In parallel circuits, total current divides inversely proportional to resistance; smaller resistance branches carry larger current
  • Total Resistance Calculation: For two parallel resistors,
    [ R_T = \frac{R_1 R_2}{R_1 + R_2} ]
  • Current Division Formula:
    [ I_1 = \frac{R_2}{R_1 + R_2} I, \quad I_2 = \frac{R_1}{R_1 + R_2} I ]
  • Multiple Parallel Resistors: Current through the (n)-th resistor is
    [ I_n = \frac{R_T}{R_n} I ]
  • Example: For 30Ω and 60Ω resistors in parallel with total current 9A, currents are 6A and 3A respectively, consistent with Kirchhoff’s Current Law
  • Voltage Calculation: Voltage across parallel resistors is the same, equal to total current times total resistance

磨課師基本電學1 3-8 分流定則

重點歸納

  • 分流定則（電流分配原理）：並聯電路中，總電流會依電阻大小分配，電阻越小的支路電流越大，電阻越大的支路電流越小
  • 總電阻計算：兩個並聯電阻的總電阻為
    [ R_T = \frac{R_1 R_2}{R_1 + R_2} ]
  • 電流分配公式：
    [ I_1 = \frac{R_2}{R_1 + R_2} I, \quad I_2 = \frac{R_1}{R_1 + R_2} I ]
  • 多個並聯電阻：第 (n) 個電阻的電流為
    [ I_n = \frac{R_T}{R_n} I ]
  • 實例計算：30歐姆和60歐姆並聯，總電流9A，分別分得6A和3A，符合克希荷夫電流定律
  • 電壓計算：並聯電阻兩端電壓相同，等於總電流乘以總電阻
