PCB Manufacturing Process Guide - Detailed Steps

Posted on 2024-03-23 Edited on 2025-03-24 In Electronics Views: Views: Disqus:

PCB Manufacturing Process Guide

Key Points (English)

  1. Initial Setup

    • Power on the engraving machine (located at the back)
    • Open PCB Prototype software
    • Check file format (AD or PRO)
    • Create new file
    • Select AD engraving area
    • Choose project item from engineering

  2. File Management

    • Two main formats: AD and K
    • AD files for circuit diagrams
    • K files for carving paths
    • File selection by number (e.g., 120220)
    • Path selection (second option)
    • File format: xxx.Protel -GTL

  3. Operation Process

    • Select forming data (4th option from left in top toolbar)
    • Use Page Up (+) to zoom in
    • Use Page End (-) to zoom out
    • Right-click to cancel forming data function
    • Right-click to frame select
    • Left-click to move view
    • Select outer frame (blue line) and inner circles (holes)
    • All selected items turn white

  4. Processing Steps

    • Select forming (5th option from left in top toolbar)
    • Outer frame and inner holes turn into blue circles
    • Outer frame turns green when outermost layer selected
    • Inner holes turn green when innermost layer selected
    • Right-click after green selection
    • Click “Calculate Engraving Logic” (6th option from left)
    • Click “Flip” (7th option from left)
    • Note: Large copper board for single-sided printing
    • Note: Small copper board for double-sided printing

  5. Alignment Process

    • Click “H” (1st option in 2nd row from right)
    • Adjust board position within copper board
    • Right-click for rotation
    • Frame selection turns blue
    • Right-click to rotate
    • Must align within white lines on sides and orange frame at bottom
    • Click “H” again
    • Board alignment: 12.15 on both sides
    • Must have sticker attached

  6. Processing Area Setup

    • Select auto correction engraving area
    • Select processing area (3rd to 6th options in 2nd row from right)
    • Note: Report if area exceeds limits
    • Select auto tool change mode (2nd to last option in 2nd row from right)
    • Set positioning hole drilling
    • Forming tool: 1.4mm
    • Wait for 1 hour

PCB Board Flipping Process

Key Points (English)

  1. Initial Setup

    • Select the 8th option from left in function bar for board flipping
    • Board needs to be flipped accordingly
    • Reapply adhesive tape

  2. CCD Alignment Process

    • Click CCD (last option in first row from right in side toolbar)
    • Use auto-alignment until CCD shows “PASS”
    • Click auto tool change processing mode
    • Select surface detection and carving path
    • Wait for 30 minutes

  3. Drilling Process

    • Use auto tool change mode (second to last option in second row from right)
    • Select “PCB Internal Drilling”
    • Size settings:
      • T1: Depth 2.8mm (standard size)
      • Second drill: 0.4mm (manual change)
      • Place in NO1 = T1
      • Occasionally switches to T2/T3
    • Note: Use simulation tool for holes over 1.5mm
    • Wait for 1 hour

  4. Final Processing

    • Use auto tool change processing mode
    • Select outer frame forming
    • Only change drill bits
    • Auto tool change setting OK
    • Recommended forming tool: 1.4mm

Important Notes

Key Points (English)

  1. Tool Selection

    • Standard depth: 2.8mm
    • Secondary drill: 0.4mm
    • Forming tool: 1.4mm recommended
    • Simulation tool for holes > 1.5mm

  2. Processing Times

    • Surface detection: ~30 minutes
    • Drilling process: ~1 hour
    • Total process time varies by board size

  3. Quality Control

    • Ensure CCD shows “PASS”
    • Check surface detection results
    • Verify drilling depths
    • Monitor outer frame formation

過程二

  1. 初始設置

    • 功能欄從左至右第8個選擇 “”翻面”””
    • 板子需要跟著側翻
    • 重貼交帶

  2. CCD對位流程

    • 點擊CCD（右側方工具欄從右數第一排倒數第一個功能）

    • 點自動對位直到CCD檢測有PASS

    • 點自動換刀加工模式

    • 點平面檢測與雕刻路線

過程三

  1. 鑽孔流程

    • 使用自動換刀模式（右側方工具欄從右數第二排倒數第二個功能）
    • 選擇鑽孔 PCB內
    • 尺寸設定：
      • T1：深度 2.8（尺寸）（一定，常規標準）
      • 第二個 0.4（手動換成0.4）
      • 放置NO1 = T1
      • （偶爾會換到T2T3）
    • 注意：超過1.5mm使用模擬刀

過程四

  1. 最終處理
    • 使用自動換刀加工模式（右側方工具欄從右數第二排倒數第二個功能）
    • 選擇外框成形
    • 只換鑽頭？？？？？？？
    • 自動換刀設定OK
    • 建議成形刀：1.4mm

重要注意事項（中文）

  1. 工具選擇

    • 標準深度：2.8mm
    • 第二鑽頭：0.4mm
    • 建議成形刀：1.4mm
    • 超過1.5mm使用模擬刀

  2. 處理時間

    • 表面檢測：約30分鐘
    • 鑽孔流程：約1小時
    • 總處理時間依板子大小而定

  3. 品質控制

    • 確保CCD顯示”PASS”
    • 檢查表面檢測結果
    • 驗證鑽孔深度
    • 監控外框成形
