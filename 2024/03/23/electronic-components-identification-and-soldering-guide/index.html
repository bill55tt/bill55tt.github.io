Electronic Components Identification and SMD Soldering Guide

Posted on 2024-03-23 Edited on 2025-03-24 In Electronics Views: Views: Disqus:

Electronic Components Identification

Resistors

  • Flat components ranging from 603 to 25 sizes
  • Have numbered markings on the surface
  • Used for current limiting and voltage division

Capacitors

  • Two-container appearance for power storage
  • Available in sizes like 603, 0806, 10
  • Gray appearance with no polarity
  • Used for filtering and energy storage

Inductors

  • Large brown square blocks
  • Easy to identify
  • No polarity
  • Used for filtering and energy storage

Diodes

  • Triangular marking on the component
  • Polarity sensitive
  • Used for rectification and protection

Transistors

  • Shiny pins
  • Common types include M-type transistors
  • Three pins marked as SPS
  • First pin connects to voltage and ground
  • Available in both small and large module sizes
  • M-type transistors have paired pins at the bottom

Inductors (Large Type)

  • Very large components
  • Easy to identify by size
  • Smaller inductors available upon request

PCB Software Operation Guide

File Types and Setup

  • Two main file formats: AD and K
  • AD files a、e primarily used for circuit diagrams
  • For AD files, create new white data
  • Navigate through component interface options
  • K files are typically in “New Carving Path”
  • AD files are in “AD Carving Path” or “AD Carving Method”

Operation Steps

  1. File Selection

    • Find file by number (e.g., 119296921)
    • Always select the second option
    • Files are provided and placed in the system

  2. Path Selection

    • Select second option for all paths
    • One for J, one for viewing
    • Terminal also uses second option
    • New items also use second option

  3. Data Selection

    • Click “Select Data” and “Select Forming Data”
    • Mouse cursor changes to four arrows
    • Click HG up to zoom
    • Select blue line around PCB
    • Select inner blue line for holes

  4. Processing

    • Click outside first (no holes)
    • Select fifth option for data offset
    • Outer frame becomes white
    • Click fifth option for fixed posts
    • Becomes two circles
    • Outer frame stores outermost
    • Changes to green
    • Right-click after green
    • Click 1-2-3-4-5, sixth is “Calculate Path”
    • Click after calculation

Panel Types

  • Single-sided panel: larger size
  • Double-sided panel: smaller size
  • Can be distinguished by appearance
  • Single panel has one side
  • Double panel has small features on top

Positioning and Calibration

  1. Use “Add Area” for positioning

    • Click four corners
    • Check upper and lower positions
    • Adjust if board is too large
    • Confirm processing mode

  2. Positioning Process

    • Wait for CDC
    • Auto-move to hole alignment
    • P indicates success
    • NG requires manual adjustment
    • Press C for next step
    • Auto tool change processing mode

  3. Surface Detection

    • Perform surface detection
    • Check carving depth
    • Proceed to line carving
    • Check for obvious lines indicating problems
    • Process takes about 10:30 to complete
    • Larger boards take longer

SMD Soldering Guide (1206 Resistor Example)

Temperature Settings

  • Exhaust fan used above 350-360°C
  • Temperature varies by component
  • Higher temperature (400°C+) for faster soldering
  • Lower temperature for slower, more controlled soldering

Basic Steps

  1. Start with 1206 size (about R6R7 edge size)
  2. Solder melts at high temperature
  3. Use dog bone-shaped soldering tip
  4. Always tin the pad first
  5. Use tweezers to hold component
  6. Align and solder one side first
  7. Solder the other side
  8. Initial solder amount may be more
  9. Ensure component is flat on pad
  10. Remove excess solder if needed

Advanced Techniques

  • Place iron close to pad
  • Hold position when heating
  • Use tweezers to hold component
  • Rotate 90 degrees if needed
  • Add flux if soldering is difficult
  • Flip board if needed
  • Heat and add solder simultaneously
  • Clean iron tip with wet sponge
  • Use small paper cup with water
  • Scrape old solder before adding new

Common Issues and Solutions

  • Excess solder: scrape or add new solder
  • Dull iron tip: replace tip
  • Connected solder: heat and press down
  • Dropped components: use tweezers on both sides
  • Keep sponge moist
  • Clean pad before soldering
  • Check for green frame on PCB
  • Ground connection for 0603 resistors without green frame

Safety and Maintenance

  • Place iron on stand when not in use
  • Don’t place directly on table
  • Clean excess solder gradually
  • Add new solder if needed
  • Replace tip when it loses shine
  • Keep sponge wet
  • Take breaks as needed

電子元件辨識指南

電阻

  • 從 603 到最大的 25 尺寸都很平
  • 零件上面會有編號
  • 用於限流和分壓

電容

  • 外觀像是兩個容器，用於儲存電力
  • 規格有 0603、0806、1206、1210 等
  • 外觀是兩個灰色的樣子
  • 沒有正負極之分
  • 用於濾波和能量儲存

電感

  • 看起來就是一個很大顆的茶色方塊
  • 容易辨識
  • 沒有正負極之分
  • 用於濾波和能量儲存

二極體

  • 在零件上會有三角形的標識
  • 有極性之分
  • 用於整流和保護

電晶體

  • 腳位通常是閃亮的
  • 最常用的是 M 型電晶體
  • 有三個腳位，標示為 SPS
  • 第一個腳位連接電壓和接地
  • 有大尺寸和小尺寸模組
  • M 型電晶體的配對腳位在底部

電感（大尺寸）

  • 超大尺寸元件
  • 容易從大小辨識
  • 小尺寸電感可向工作人員詢問

SMD 焊接指南（以 1206 電阻為例）

溫度設定

  • 抽風機在 350-360°C 以上使用
  • 溫度依零件而定
  • 較高溫度（400°C+）用於快速焊接
  • 較低溫度用於較慢、較控制的焊接

基本步驟

  1. 從 1206 尺寸開始（約 R6R7 邊緣大小）
  2. 焊錫在高溫下融化
  3. 使用狗骨頭狀烙鐵頭
  4. 先在焊盤上上錫
  5. 用鑷子夾住零件
  6. 對準並先焊一邊
  7. 再焊另一邊
  8. 初始焊錫量可能較多
  9. 確保零件平貼在焊盤上
  10. 如有需要去除多餘焊錫

進階技巧

  • 將烙鐵靠近焊盤
  • 加熱時保持位置
  • 用鑷子夾住零件
  • 需要時可旋轉 90 度
  • 焊接困難時可加助焊劑
  • 需要時可翻轉板子
  • 同時加熱和加焊錫
  • 用濕海綿清潔烙鐵頭
  • 使用裝水的小紙杯
  • 加新焊錫前刮除舊焊錫

常見問題和解決方案

  • 焊錫過多：刮除或加新焊錫
  • 烙鐵頭無光澤：更換烙鐵頭
  • 焊錫相連：加熱並下壓
  • 零件掉落：用鑷子從兩側夾住
  • 保持海綿濕潤
  • 焊接前清潔焊盤
  • 檢查 PCB 上的綠色框
  • 沒有綠色框的 0603 電阻可能需要接地

安全和維護

  • 不使用時將烙鐵放在架上
  • 不要直接放在桌上
  • 逐漸清理多餘焊錫
  • 需要時加新焊錫
  • 烙鐵頭失去光澤時更換
  • 保持海綿濕潤
  • 適時休息
